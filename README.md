# 🍃 EcoPulse AI — Interactive Carbon Footprint Tracker & Sustainability Platform

EcoPulse AI converts abstract ecological metrics into dynamic, interactive landscapes. Calculate your carbon footprint, get personalized guidance from an AI Sustainability Coach, and watch your virtual forest heal as you commit to carbon-reducing habits.

🚀 **Live Demo:** [https://ecopulse-ai-lime.vercel.app/](https://ecopulse-ai-lime.vercel.app/)

---

## ✨ Features

- **🌍 3D Earth Globe Visualizer:** An interactive, canvas-rendered globe that reflects the health of the planet based on your real-time carbon score.
- **📊 Carbon Footprint Calculator:** An intuitive, multi-step diagnostic tool evaluating your daily transit, diet, home energy, and shopping habits.
- **🤖 AI Sustainability Coach:** An intelligent companion that audits your choices and suggests actionable, personalized improvements.
- **🌲 Virtual Digital Forest:** A gamified ecosystem that grows, blooms, or decays dynamically based on your habits and claimed eco-challenges.
- **🏆 Gamified Eco Challenges & Leaderboard:** Complete weekly tasks, earn XP, level up, and compare your impact on the global community leaderboard.
- **📈 Impact Visualizer & Journey Timeline:** Beautiful Chart.js graphs mapping carbon offsets and tracking your green timeline over months.
- **🗺️ Global Carbon Intensity Map:** Visualizes regional grid intensity to show where energy clean-up is needed most.

## 🛠️ Tech Stack

EcoPulse AI is built using modern frontend technologies optimized for visual excellence:

- **Framework:** [Next.js](https://nextjs.org/) (React 19)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/) (GreenSock) for high-performance micro-interactions
- **Charts & Data:** [Chart.js](https://www.chartjs.org/) & [React ChartJS 2](https://react-chartjs-2.js.org/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **3D & Canvas Rendering:** HTML5 Canvas & custom SVG shaders

## 🎨 Design System & Aesthetics

The application features a curated, visually immersive aesthetic:

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.x or higher)
- npm, yarn, pnpm, or bun

### Local Installation

1. Clone the repository:
   ```bash
   git clone <YOUR_GITHUB_REPO_URL>
   cd ecopulse-ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3001](http://localhost:3001) in your browser to view the application.
